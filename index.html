Height formula example

Total height = cargo height + flat rack overhang + chassis height
Example: 115" + 24" (40'FR) + 48" (std chassis) = 187" — TOO HIGH (over 180" max)
Fixed: 115" + 24" (40'FR) + 38" (PUMA) = 177" — GOOD (under 180")
